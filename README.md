# my-golang
Belajar Golang

## Inisialisasi Proyek Golang:
### Di dalam direktori proyek, inisialisasi modul Go:
```sh
go mod init github.com/Gopartner/my-golang
```
## Membuat file:
```sh
vim main.go
```
## Tambahkan kode berikut sebagai permulaan:
```sh
package main

import "fmt"

func main() {
    fmt.Println("Welcome to my Golang project!")
}
```

## Menjalankan Program:
```sh
go run main.go
```
